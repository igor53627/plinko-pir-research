# Setting Up GitHub for Plinko PIR Research

This repository was extracted from a larger research monorepo to focus on Plinko PIR implementation.

## Option 1: Create New GitHub Repository

```bash
cd ~/pse/plinko-pir-research

# Create new repo on GitHub first at https://github.com/new
# Then:

git remote add origin https://github.com/YOUR_USERNAME/plinko-pir-research.git
git branch -M main
git push -u origin main
```

## Option 2: Push to Existing GitHub Repo

```bash
cd ~/pse/plinko-pir-research

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main
```

## Git LFS Setup (Required for Binary Files)

This repo uses Git LFS for large binary files (hint.bin, database.bin, address-mapping.bin).

```bash
# Install Git LFS (if not already installed)
brew install git-lfs

# Initialize Git LFS in the repository
cd ~/pse/plinko-pir-research
git lfs install

# The .gitattributes file is already configured to track *.bin files
# To push the actual LFS files to GitHub:
git lfs push origin main --all
```
